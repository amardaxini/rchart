    create_color_gradient_palette(r1,g1,b1,r2,g2,b2,shades) click to toggle source

    Create a color palette shading from one color to another This function will fill the color palette with 10 shades between the two RGB colors 0,0,0 and 100,100,100.This will produce grey shades. (Palette id 0-9 will be filled)

    # File lib/color_palette.rb, line 12
def create_color_gradient_palette(r1,g1,b1,r2,g2,b2,shades)
  r_factor = (r2-r1)/shades
  g_factor = (g2-g1)/shades
  b_factor = (b2-b1)/shades
  i= 0
  while(i<= shades-1)
    set_palette(i, r1+r_factor*i, g1+g_factor*i, b1+b_factor*i)
    i = i+1
  end
end

def load_color_palette(color_palette)
  color_id = 0
  color_palette.each do |palette|
    if palette.length == 3
      set_palette(
          color_id, palette[0].to_i, palette[1].to_i, palette[2].to_i)
      color_id+=1
    end
  end
end
    load_color_palette_from_file(file_name) click to toggle source

    This function will load the color scheme from a text file. This file must be formatted with three values per line ( r,g,b ). By default the delimiter is a coma but you can specify it.

    # File lib/color_palette.rb, line 26
def load_color_palette_from_file(file_name)
  color_id = 0
  File.open(file_name,"r") do |infile|
    while (line = infile.gets)
      values = line.split(",")
      if ( values.length == 3 )
        set_palette(color_id, values[0].to_i, values[1].to_i, values[2].to_i)
        color_id+=1
      end
    end
  end
end
